The extension connects to all of these at once, so different MCP hosts (e.g. Claude Code on 8765, Claude Desktop on 8766) can each drive it — one at a time.
When on, the tab Claude is acting on becomes the active, visible tab and its window is focused.
A visible pointer animates to each click so you can watch Claude work (also appears in screenshots).

Setup

  1. Run the MCP bridge (Claude Code launches it automatically once added via claude mcp add).
  2. Make sure the port above matches the bridge's port (default 8765).
  3. The status dot turns green when the extension is connected to the bridge.
